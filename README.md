# py_word_test

### exe file
파이썬 버전문제, path 해결 후<br>
pip install pyinstaller<br>
pyinstaller -F -w wordTest.py<br>
-F 옵션을 통해 단일파일 생성<br>
-w 옵션을 통해 실행 시 터미널 뜨지 않도록 적용<br>

(주의) 해당 csv 파일이 같은 폴더에 있어야 정상작동됨
